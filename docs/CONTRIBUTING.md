# Contributing to Grap3

We love crypto-native builders. 🍇

## Ground rules
- This repo is original MIT-licensed code. Do NOT paste proprietary code from
  other dating apps (Tinder/Bumble/etc.) — patterns/ideas only.
- Never commit secrets, keypairs, or .env files.

## Workflow
1. Fork + branch: feat/your-feature
2. Keep PRs focused and described
3. Run lint before pushing
4. One approving review from a maintainer to merge

## Areas we need help

- Seeker-specific UI polish
